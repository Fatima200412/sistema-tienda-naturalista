# sistema-tienda-naturalista

Proyecto final de Programación Orientada a Objetos - CUCEI

## Descripción

Sistema simple para gestionar inventario y ventas de una tienda naturista.

## Compilar

```bash
g++ -o tienda main.cpp implementacion/*.cpp -Iheaders
```

## Ejecutar

```bash
./tienda
```

## Funcionalidades

- Gestión de productos (CRUD: Capturar, Mostrar, Eliminar, Buscar, Modificar, Ordenar, Consultar)
- Registro de ventas
- Alertas de stock bajo (≤3 unidades)
